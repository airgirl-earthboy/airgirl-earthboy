# Airgirl and Earthboy

Help Airgirl and Earthboy explore the industrial wastelands!

## About

2D co-op game where players work together to navigate mazes and unlock levels, inspired by [Fireboy and Watergirl](https://www.coolmathgames.com/0-fireboy-and-water-girl-in-the-forest-temple)

## How to Play

- Use A.W.D to move **Airgirl**

- Use arrows to move **Earthboy**

- **Goal:** Collect all gears to clear level

[**Play now!**](https://airgirl-earthboy.github.io)

## Local Development

### Prerequisites

- Unity [Hub](https://unity.com/download)

- Unity [6000.0.58f2](https://unity.com/releases/editor/whats-new/6000.0.58f2)

### How to Run 

1. Open the Unity Hub, and go to the `Projects` tab

2. Click the `New Project` button

3. Select `6000.0.58f2` for the editor version, and then choose `Universal 2D` for the template

4. Give the project a name of your choosing, select a location for the project, and click `Create project`

5. Open a terminal once the project has finished loading, and clone this repository with Git

    ```bash
    git clone https://github.com/airgirl-earthboy/airgirl-earthboy.git
    ```

6. Open your file explorer, and then locate the `airgirl-earthboy` directory of the cloned repository

7. Move the `airgirl-earthboy` folder to the `Assets` folder of your newly created Unity project

8. Return to your open Unity project, wait for it to finish loading, and press the play button at the top to run the game
