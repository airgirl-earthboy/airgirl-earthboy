# Airgirl and Earthboy

Help Airgirl and Earthboy explore the industrial wastelands!

## About

Airgirl and Earthboy is a **2D co-op game** inspired by [Fireboy and Watergirl](https://www.coolmathgames.com/0-fireboy-and-water-girl-in-the-forest-temple) where players work together to navigate mazes and unlock levels.

Airgirl and Earthboy was originally created as a submission for the **Video Game Design** event of the [Technology Student Association](https://tsaweb.org/), a national organization that hosts STEAM-related competitions. A Scratch version of this game was an entry for the 2025 [Gulf Coast Regional Conference](https://www.gulfcoasttsa.org/) of [Texas TSA](https://texastsa.org/), and this Unity version of the game was made as part of an end-of-year project for a game development high school class.

## How to Play

- Use A.W.D to move **Airgirl**

- Use arrows to move **Earthboy**

- **Goal:** Collect all gears to clear level

[**Play now!**](https://airgirl-earthboy.github.io)

## Behind the Website

Airgirl and Earthboy was created on Unity 6 using C#. For our [TSA](https://tsaweb.org/) submission, we had to create a multiplayer game for 2 to 4 players that was playable on one shared keyboard and mouse. When we began the process of developing our game, we decided to use Scratch to create a playable prototype since we did not have much game developing experience at the time.

Months later, when I was assigned my end-of-year project for my Unity class, I decided that this project was the perfect opportunity for me to remake the game I had originally started with my TSA team and put the Unity skills I had learned throughout the year to good use. At first, it was a bit difficult for me to figure out how to start translating the Scratch version to Unity, but after breaking up the game into smaller chunks that were easier to work on, writing some pseudocode to plan out the entire game, and prioritizing the essential game components for a minimum viable product (start menu, moving characters, level with obstacles), I was able to recreate the game on Unity with even more features than the original Scratch version.

## Local Development

### Prerequisites

- Unity [Hub](https://unity.com/download)

- Unity [6000.0.58f2](https://unity.com/releases/editor/whats-new/6000.0.58f2)

### How to Run 

1. Open the Unity Hub, and go to the `Projects` tab

2. Click the `New Project` button

3. Select `6000.0.58f2` for the editor version, and then choose `Universal 2D` for the template

4. Give the project a name of your choosing, select a location for the project, and click `Create project`

5. Open a terminal once the project has finished loading, and clone this repository with Git

    ```bash
    git clone https://github.com/airgirl-earthboy/airgirl-earthboy.git
    ```

6. Open your file explorer, and then locate the `airgirl-earthboy` directory of the cloned repository

7. Move the `airgirl-earthboy` folder to the `Assets` folder of your newly created Unity project

8. Return to your open Unity project, wait for it to finish loading, and press the play button at the top to run the game
